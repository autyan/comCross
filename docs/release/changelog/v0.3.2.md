# v0.3.2 Changelog

## Release Goal

Improve day-to-day usability after v0.3.1 by refining the Shell experience,
session management, localization behavior, and Linux serial port handling.

## User-Facing Changes

- Editable session names.
- Notification management improvements.
- Language selection follows the system language more reliably.
- Graceful exit flow with asynchronous cleanup and progress reporting.
- TX/RX color improvements and more compact message display.
- Tool tabs use clearer icon treatment.
- Sidebar dropdown, refresh, and statistics interactions were improved.
- Font customization and connection behavior were refined.
- Settings UI was compacted.

## Linux Serial Port Handling

- Added Linux serial scan pattern and exclude-filter support.
- Added Linux permission handling through privilege escalation where needed.
- Improved serial permission error handling and user feedback.
- Added documentation for serial port permission management.

## Technical Changes

- Added or refined workspace, export, message box, and command helper services.
- Improved reactive updates in Shell models.
- Improved localization resource usage in the Shell.
- Fixed ComboBox binding and tool tab state issues.
- Fixed language combo state, auto-scroll behavior, and session state
  persistence issues.

## Documentation

- Added persistence documentation.
- Added serial port permission management documentation.

## Compatibility

- No breaking migration step is required when upgrading from v0.3.1.
- v0.4.0 supersedes the v0.3 development line as the first stable baseline.

## Known Limitations

- Package signing and the current v0.5 release verification flow are not part
  of this release.
- The plugin and session architecture is substantially converged in v0.4.0.
