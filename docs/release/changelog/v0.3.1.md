# v0.3.1 Changelog

## Release Goal

Improve application identity and desktop integration while preparing early
Windows and Linux distribution artifacts.

## User-Facing Changes

- Added application icons.
- Added Linux desktop integration metadata.
- Fixed Windows DLL/icon presentation issues.
- Improved package content layout for the Shell, plugin hosts, tools, and
  official plugins.

## Packages

- Windows portable package artifacts were provided for the v0.3.1 line.
- Linux package artifacts were provided for the v0.3.1 line, including
  framework-dependent and self-contained variants.
- MSI packaging was still planned for a later release.

## Included Official Plugins

- Serial statistics plugin.
- Serial protocol plugin.
- Serial flow plugin.

## Compatibility

- This release belongs to the v0.3 development line.
- v0.4.0 supersedes the v0.3 development line as the first stable baseline.

## Known Limitations

- MSI packaging is not available in this release.
- Package signing and the current v0.5 release verification flow are not part
  of this release.
