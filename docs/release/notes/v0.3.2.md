# ComCross v0.3.2

ComCross v0.3.2 is a usability and UX enhancement release for the v0.3 line. It
improves session management, visual feedback, settings, localization behavior,
and Linux serial port handling.

## Downloads

This historical release provides the packages attached to the original GitHub
Release, including portable archives and Linux package artifacts from that
release line.

## Important Notes

- No special migration step is required when upgrading from v0.3.1.
- Package signing and the current v0.5 release verification flow are not part
  of this release.
- v0.4.0 replaces this development line as the first stable baseline.
