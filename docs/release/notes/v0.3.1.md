# ComCross v0.3.1

ComCross v0.3.1 focuses on application icons, desktop integration, and early
distribution packaging for Windows and Linux.

## Downloads

This historical release provides the packages attached to the original GitHub
Release, including Windows portable archives and Linux package artifacts from
that release line.

## Important Notes

- MSI packaging was not available in this release.
- Package signing and the current v0.5 release verification flow are not part
  of this release.
- v0.4.0 replaces the v0.3 development line as the first stable baseline.
