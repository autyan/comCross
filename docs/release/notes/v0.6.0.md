# ComCross v0.6.0

ComCross v0.6.0 improves long-running message work with session archive
history, a stronger message viewer, cancellable search, and the first official
static website.

## Downloads

Choose the package for your system:

- Windows: MSI for x64 or ARM64
- Ubuntu / Debian: DEB for x64 or ARM64
- Fedora: RPM for x64 or ARM64
- Other Linux distributions: AppImage for x64 or ARM64

## Verification

Verify downloaded files with:

```bash
sha256sum -c SHA256SUMS
gpg --verify SHA256SUMS.asc SHA256SUMS
```

## Important Notes

- Session Archive is optional and remains a session-level user choice.
- Archive history mode is for browsing stored frames, not replay or archive
  import.
- Search is a message-viewer navigation aid. Persistent full-text indexing,
  decoded-field search, and cross-workspace search are not part of v0.6.0.
- Plain, Slim, and Detailed message display modes now have distinct rendering
  rules. Detailed mode keeps frame structure visible; Plain and Slim prioritize
  text selection and stream reading.
- macOS packages are not provided in this release.
- The current Windows signing certificate is a self-signed test code-signing
  certificate.
