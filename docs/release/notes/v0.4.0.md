# ComCross v0.4.0

ComCross v0.4.0 is the architecture stabilization release. It establishes the
main program and bus plugins as a producer/consumer boundary and makes the TCP,
UDP, Serial, session, and message flows stable enough for wider use.

## Downloads

This release provides portable Windows and Linux archives for x64 and ARM64.
Installable signed packages are introduced in v0.5.0.

## Important Notes

- This is the first stable baseline after the v0.3 development line.
- Legacy workspace/session migration from old v0.3 development state was
  removed. Back up old data before upgrading from pre-v0.4 builds.
- Deleting a session deletes its associated session data.
- MSI, DEB, RPM, AppImage, signing, and automated release packaging are handled
  in v0.5.0 and later.
