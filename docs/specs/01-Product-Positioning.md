# Product Positioning

## Summary
comCross is a cross-platform toolbox for embedded engineers. The core experience is a workspace-first message stream. Tools are modular and optional, designed to surround the workspace without interrupting it.

## Target Users
- Embedded engineers working with serial devices.
- Test and validation engineers who need reliable logs and repeatable workflows.
- Field engineers who require stable, low-friction troubleshooting.

## Key Principles
- Workspace is always visible.
- Tools are peripheral and share the workspace context.
- Linux-first, consistent UI across platforms.

## Differentiation
- High-density, readable message stream.
- Modular toolset with offline plugin loading.
- Workspace state persistence as a first-class feature.
