## Plan

1) I18N 基础改造（优先级最高）
   - 资源体系：新增 Assets 资源包，内置默认英语
   - 语言切换：即时刷新 UI，不重启
   - 兼容性：保留现有 GetString/Binding 入口

2) 配置中心与通知中心
   - 状态栏入口 + 悬浮配置页
   - 通知中心 UI 与历史数据结构
   - 配置项：语言、日志、通知、会话/连接、显示/性能、导出

3) 日志自动保存与存储策略
   - 按 session + 起始时间戳命名
   - 单文件 10MB 分片
   - 总占用默认 256MB，可配置
   - 超限只通知，可选自动删除（感叹号悬浮说明规则）

4) SQLite 持久化
   - 通知中心数据
   - 日志索引/摘要
   - 配置历史

5) 插件机制基础
   - plugins 目录
   - manifest 嵌入 dll 校验
   - 插件配置由主程序 JSON 管理

6) Build 体系
   - 平台/架构/配置矩阵构建
   - 产物命名规范落地

7) 视觉与品牌
   - 现代化 icon
   - Assets 接入

## Acceptance

- i18n 切换即时生效，默认英文可用
- 配置中心与通知中心可用，配置持久化稳定
- 日志自动保存按策略切分与提示
- SQLite 覆盖通知、日志索引与配置历史
- 插件目录与校验流程可用
- Build 产物按矩阵输出
