## Translation Policy

- English source documents remain authoritative.
- Localized documents should be clear, conservative, and avoid adding claims not
  present in the English source.
- Security, signing, release verification, installation-risk, and compatibility
  wording must preserve the original meaning.
- Every localized document must include an AI-assisted documentation notice.
